Get Started

From URL From JSON
Step 1

Create your character

Recreate your characters using maplestory.studio or maples.im. Right-click and select Copy Image Address to add the URL to your clipboard
Step 2

Import your character

Type the command mush import <char_name> <url> with the name you want to use and paste the URL from your clipboard
Step 3

Set info

Optional. Use mush set info to input information others can see in your character profile
Step 4

Start mushing

All of the emote, sprite, or action commands should now be usable
