01 / emotes

Send emotes of your character using all standard emotions found in game like love, cry, shine, etc. (F1 - F7 key bindings in PC or emotes on mobile)

02 / actions

Perform an action on another discord member's character. If they don't have any characters registered, you'll target their profile picture instead

03 / profiles

Set up your character's info (IGN, server, etc.) and choose an expression/pose for others to check out. React with or to fame/defame each other
